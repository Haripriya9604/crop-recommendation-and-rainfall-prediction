About Smart Agro Assistant

A data-driven decision support system that helps farmers choose suitable crops using rainfall and soil information.

üéØ Project Objective

The main goal of Smart Agro Assistant is to support farmers and agriculture stakeholders in making better crop planning decisions by combining:

  • Recent rainfall history (3-month lagged rainfall).
  • Soil nutrients: Nitrogen (N), Phosphorus (P), Potassium (K).
  • Weather and soil conditions: temperature, humidity, and pH.

The recommendation is not meant to replace expert advice, but to give a strong, data-backed starting point.

üß† Machine Learning Models

  • Rainfall Prediction: Random Forest Regressor trained on IMD rainfall data to estimate upcoming monthly rainfall using the previous 3 months as input.
  • Crop Recommendation: Random Forest Classifier trained on the Kaggle Crop Recommendation dataset, mapping NPK + pH + temperature + humidity + rainfall to the most suitable crop.
  • Evaluation: MAE/RMSE for rainfall prediction, and accuracy + classification report for the crop model.

First, rainfall is predicted. That value is then passed into the crop model as one of the features.

  • Backend: Python, Flask, Scikit-learn, Pandas, NumPy, Joblib
  • Frontend: HTML, CSS, JavaScript, Chart.js
  • Data Sources: IMD Rainfall Dataset, Crop Recommendation Dataset (Kaggle)
  • Environment & Tools: Virtualenv, VS Code, Git, GitHub

The UI is designed to be simple for farmers, yet detailed enough for project reviews and technical discussions.
